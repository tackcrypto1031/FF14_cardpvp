# FF14 九宮幻卡模擬器 (Triple Triad Simulator & Solver)

這是一個專為《最終幻想14》（FF14）設計的高精度九宮幻卡（Triple Triad）模擬器與解法器。它不僅能幫助玩家模擬各種複雜的遊戲規則，還能提供基於 **Minimax 演算法** 的最優解建議，助你輕鬆戰勝高級 NPC。

## 核心功能

*   **牌組管理 (Deck Management)**
    *   支持最多 **10 組** 自定義牌組。
    *   提供重命名、刪除與自動保存功能（基於 `localStorage`）。
    *   **編輯模式**：直接點擊卡牌即可修改數值（1-9, A）與種族。
    *   **佈局優化**：種族圖示（蠻族、血盟等）置於卡牌正中心，確保數值清晰可見。

*   **實時敵方追蹤 (Live Enemy Tracking)**
    *   **對手手牌面板**：右側新增獨立面板，即時保存對手出過的每一張牌。
    *   **狀態顯示**：已放置在棋盤上的對手卡牌會顯示為灰色，方便判斷剩餘手牌。
    *   **拖放支持**：可直接將已知的敵方卡牌拖拽回棋盤，快速復現戰局。

*   **戰術 AI (Advanced AI Solver)**
    *   **Minimax 策略**：AI 不再僅僅考慮當前一步，而是會針對已知的敵方手牌進行 **兩步預判 (2-ply lookahead)**，模擬對手的最優應對，提供更深層次的策略建議。
    *   **回合限制**：分析按鈕會根據當前回合狀態切換，確保僅在玩家回合提供有效建議。

*   **再來一局 (Rematch Strategy)**
    *   **資訊累加**：點擊「再來一局」會保留並累加對手的卡牌資訊。通過多次對局，你可以逐步揭開 NPC 的完整牌組，建立完美的應對方案。
    *   **原樣歸還**：即便你在對局中翻轉（佔領）了對手的卡牌，「再來一局」時系統也會正確識別其初始歸屬並歸還至敵方手牌。

*   **完整規則引擎 (Rule Engine)**
    *   支持 FF14 所有核心規則：**逆轉**、**同數**、**加算**、**王牌殺手**、**上升/下降**、**順序** 等。

## 使用指南

1.  **配置牌組**：點擊左側面板的「編輯」進入編輯模式，調整你的主力陣容。
2.  **設置規則**：在右側面板勾選本次對局的特殊規則。
3.  **模擬對局**：
    *   將你的卡牌拖拽至棋盤。
    *   點擊空白格輸入敵方卡牌數值，或者從右側「對手手牌」面板直接拖拽已知卡牌。
4.  **獲取建議**：點擊「戰術分析」。如果已知對手部分手牌，AI 會表現得更加聰明且具備預判性。
5.  **長期作戰**：利用「再來一局」不斷收集對手情報，直到徹底分析出 NPC 的所有底牌。

## 安裝與運行

### 前提條件
*   安裝了 [Node.js](https://nodejs.org/) (建議 v18+)

### 快速開始
1.  **安裝依賴**:
    ```bash
    npm install
    ```
2.  **啟動開發服務器**:
    ```bash
    npm run dev
    ```

### 快捷腳本 (Windows)
*   `install.bat`: 自動安裝依賴。
*   `start-dev.bat`: 啟動開發服務。
